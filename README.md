# QRGen By AWC

A professional QR code generator with advanced customization features.

## Features

- Generate multiple QR codes from text input
- Live preview with real-time updates
- Customizable QR appearance (colors, roundness)
- Logo embedding with color matching options
- Custom captions for each QR code
- Bulk download as SVG ZIP
- Bootstrap-based responsive UI

## Usage

1. Enter values in the QR Input textarea (one per line)
2. Optionally add captions in the Captions textarea
3. Customize appearance with the controls
4. Upload a logo if desired
5. Use color matching buttons for logo-QR coordination
6. Click Generate to create QR codes
7. Download individually or bulk as ZIP

## Technologies

- HTML5, CSS3, JavaScript
- Bootstrap 5
- QR Code Styling library
- JSZip for bulk downloads
